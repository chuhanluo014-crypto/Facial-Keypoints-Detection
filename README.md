# SDSC8007 Facial Keypoints Detection

This repository contains an SDSC8007 mini project for facial keypoints detection. The project trains convolutional neural network models to predict 15 facial landmark coordinates from 96x96 grayscale face images.

## Project Contents

- `main/main.ipynb`: main experiment notebook with data loading, augmentation, baseline CNN, MobileNetV2 model, masked loss, and cross-validation training.
- `main/training.csv`: training data with facial keypoint coordinates and image pixels.
- `main/test.csv`: test image data.
- `main/IdLookupTable.csv`: Kaggle-style submission lookup table.
- `main/SampleSubmission.csv`: sample submission format.
- `SDSC8007_Final_Report.pdf`: final project report.
- `presentation.pdf`: final presentation slides.
- `description/`: project instructions and presentation template.

## Project Name

The original folder name was generic. Based on the notebook, data files, and report materials, this project is named **SDSC8007 Facial Keypoints Detection**.

## Environment

Create an isolated Python environment before running the notebook.

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

If you need a CUDA-enabled PyTorch build, install PyTorch using the command generated by the official PyTorch selector, then install the remaining packages from `requirements.txt`.

## Data And Large Files

Large CSV files and binary report assets are tracked with Git LFS. After cloning this repository, install Git LFS and pull the tracked files:

```powershell
git lfs install
git lfs pull
```

## Running

Open `main/main.ipynb` in Jupyter or VS Code and run the notebook cells from top to bottom. The notebook expects the CSV files to be available in the same directory as the notebook, so run it from `main/` or keep the existing paths.

## Model Summary

The notebook includes:

- A simple CNN baseline for 30-coordinate regression.
- A MobileNetV2-based model adapted for single-channel grayscale input.
- Masked MSE loss to handle missing keypoint annotations.
- RMSE evaluation with 5-fold cross-validation.
